# Advanced Password Strength Analyzer Web

A modern, animated web app for advanced password analysis using Flask (Python) and a responsive HTML/CSS/JS frontend.

## Features
- Animated, responsive UI (Bootstrap 5)
- Password entropy, brute force, dictionary, pattern, and breach checks
- Animated progress bars and transitions
- REST API backend (Flask)

## Usage
1. Install requirements: `pip install -r requirements.txt`
2. Run: `python app.py`
3. Open http://127.0.0.1:5000 in your browser
